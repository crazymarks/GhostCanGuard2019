# GhostMuseumSuperGuard2019
tgs2019用ゲームです、原案の名前はGhostCanGuard

殺人幽鬼、超能力を持つ警備員、そして無力な泥棒がこの博物館に
プレイヤーは警備員をコントロールし、館内の一番重要な宝を守る
しかし、殺人鬼は皆を殺す
殺人事件が起こったら、解雇されてしまう
己の仕事を守るために、館内の展示品を超能力で操り、
殺人鬼の手から泥棒を守り、そして捕まえる
ゲームです！

このゲームの開発環境は　Unity2018.3.8f1
